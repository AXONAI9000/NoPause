# NoPause

**切换标签页或鼠标焦点变化时，视频不再自动暂停。**

---

## 解决什么问题？

很多视频网站会在你切换到其他标签页时自动暂停视频。这在以下场景很烦人：

- 边看视频边查资料
- 听视频当背景音乐
- 多任务处理时想继续播放
- 查看某些网站，想边听声音，边找更好看的视频时

NoPause 让你重新掌控播放权。

## 支持的网站

理论上支持所有网站，包括但不限于：

- MissA...
- 其他使用 Page Visibility API 检测的网站

## 安装

1. 下载本项目
2. 打开 Chrome或Edge，访问 `chrome://extensions/` 或 `edge://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

## 使用

1. 访问视频网站
2. 点击浏览器工具栏的 NoPause 图标
3. 打开开关
4. 刷新页面（首次启用需要）

开关状态会按网站记住，下次访问自动生效。

## 原理

通过拦截 `visibilitychange` 事件和重写 `document.hidden` / `document.visibilityState` 属性，让网页以为标签页始终处于前台。
